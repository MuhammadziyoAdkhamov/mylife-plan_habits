# MYLife Plan Design Handoff

The implementation is based on the included `reference.png`.

Core visual direction:
- Premium dark navy/black background.
- Purple/indigo gradient as primary brand color.
- Cyan, emerald, and gold accents for progress, success, and XP.
- Glassmorphism cards with subtle borders.
- Rounded UI, large spacing, readable Poppins typography.
- Dashboard-first UX with progress score, habits, XP, streaks, journeys, and badges.

Important screens implemented:
1. Splash
2. Onboarding page flow
3. Login / sign-up UI
4. Dashboard
5. Add habit
6. Habit detail
7. Statistics
8. 30-day journey
9. XP & level
10. Badges
11. Profile
12. Settings sections

This is a local-first portfolio version. Backend sync can be added later by replacing `LocalStorageService` with repository interfaces for Firebase or Supabase.
